## Banking Application
A backend banking application built using Java and Spring Boot that provides
secure REST APIs for managing users, accounts and transactions.

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Spring Security

## Features
- User registration and authentication
- Account creation
- Deposit and withdrawal
- Fund transfer between accounts
- Transaction history
- Secure APIs using Spring Security
  
## Project Architecture
- Controller Layer
- Service Layer
- Repository Layer

## How to Run
- Clone the repository
- Configure MySQL database in application.properties
- Run the project using your IDE or mvn spring-boot:run


##  Project Evolution
This project is continuously evolving to simulate real-world banking operations.
